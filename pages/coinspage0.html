Top 10 Criptomoedas por Capitalização de Mercado



Bitcoin

Primeira criptomoeda, responsável pela propagação da tecnologia blockchain no mundo. Usada como moeda e reserva de valor.

Projeto Código-fonte Documentação

Ethereum

Primeiro blockchain “progamável”, ou seja, permite criação de contratos inteligentes e aplicações descentralizadas a partir de uma linguagem de programação própria (solidity).

Projeto Código-fonte Documentação

Ripple

Conecta bancos e provedores de pagamentos, servindo como alternativa às transferências SWIFT tradicionais.

Projeto Código-fonte Documentação

Bitcoin Cash

Fork do Bitcoin. Tenta trazer escalabilidade aumentando o tamanho do bloco de 1 MB para 8 MB.

Projeto Código-fonte

Litecoin

Uma das mais antigas criptomoedas, criada com o objetivo de ser mais rápida que o Bitcoin. Foi a primeira moeda a usar Scrypt como função hash.

Projeto Código-fonte Documentação

EOS

Plataforma para smart contracts com objetivo de ser mais escalável e fácil de utilizar. Diferenças em relação à Ethereum: disponibiliza kit de ferramentas web, interfaces auto-explicativas, usa delegated proof of stake e roda a partir de qualquer linguagem Web Assembly.
